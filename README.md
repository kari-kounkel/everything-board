# The Everything Board 🔮

**For people who do everything.**

A marble-and-binder project management system for creative multi-hyphenates who refuse to be one thing.

## Features

- **The Marble Bag** — interactive dashboard with universe marbles sized by project count
- **User-Created Universes** (3–13) — name, color, emoji, overlap visualization
- **15 Binder Tabs** — organize across shows, books, devotionals, business, and more
- **Cross-Board Kanban** — click any marble to see all cards for that universe
- **Card Linking** — depends on, feeds into, part of, related to
- **CSV Import/Export** — full round-trip with template
- **Drag & Drop** — move cards between status columns
- **Master To-Do** — weekly planner with day columns and icebox

## Quick Start

```bash
npm install
npm run dev
```

## Deploy

Push to GitHub → auto-deploys via Vercel.
